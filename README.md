# 🍿 Netflix-Style AI AWS SAA Coach

An interactive, AI-powered AWS Solutions Architect Associate (SAA) learning tool built using AWS PartyRock.

This project transforms traditional AWS exam preparation into a **Netflix-style learning experience**, where users actively engage with questions and learn through explanations instead of passive memorization.

## 💡 Why I Built This

Studying for AWS SAA often feels repetitive:
- Practice tests
- Memorization
- Passive learning

But the real exam is about **multi-service, real-world scenarios**, not just definitions.

So I asked myself:

> What if AWS learning felt like Netflix — interactive, engaging, and addictive?

---

## ⚙️ What This Project Does

This AI learning app:

- Generates AWS SAA-style questions  
- Supports **Beginner + Intermediate levels**  
- Provides instant AI explanations  
- Teaches WHY answers are correct, not just right/wrong  
- Simulates real exam-style thinking  

---

## 🎬 How It Works

1. User selects difficulty (Beginner / Intermediate)  
2. AI generates AWS scenario-based question  
3. User selects an answer  
4. AI explains:
   - Why correct answer works  
   - Why others are wrong  
   - AWS concept behind it  

---

## 🧠 Key Learning Idea

Instead of memorizing AWS services, users learn:

- How services interact  
- How to think like an AWS architect  
- How to solve real-world scenarios
